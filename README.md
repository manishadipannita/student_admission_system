# 🎓 Student Admission System (PHP + MySQL)

## 📌 Project Overview

The Student Admission System is a web-based application developed using PHP and MySQL. It allows users to manage student admission records through a simple and interactive interface.

---

## 🚀 Features

* Student admission form
* Add new student records
* View all students
* Delete student records
* File upload support
* Database integration (MySQL)

---

## 💻 Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* XAMPP (Local Server)

---

## 📂 Project Structure

* `index.php` → Home page
* `admission.php` → Admission form
* `students.php` → Student list page
* `save_student.php` → Save data to database
* `delete.php` → Delete student record
* `db.php` → Database connection
* `database.sql` → Database structure
* `uploads/` → Stores uploaded files

---

## ⚙️ How to Run This Project

1. Install XAMPP
2. Copy project folder to `htdocs`
3. Import `database.sql` into phpMyAdmin
4. Start Apache & MySQL
5. Open browser:

   ```
   http://localhost/your-folder-name/index.php
   ```
